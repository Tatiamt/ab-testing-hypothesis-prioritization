\# A/B Testing and Hypothesis Prioritization



This project analyzes business hypotheses and the results of an A/B test conducted for an e-commerce platform.



\## Project Goals



\- Prioritize hypotheses using ICE and RICE frameworks

\- Analyze A/B test performance

\- Detect and filter outliers

\- Apply statistical tests (Mann–Whitney U)

\- Recommend the best business decision



\## Tools



\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- SciPy



\## Final Result



Group B showed a statistically significant improvement in conversion rate while average order value remained stable.  

Based on the analysis, version \*\*B\*\* should be implemented.
